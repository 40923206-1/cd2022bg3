大組網頁同步方法

方法一:

1.黑窗先進入本地倉儲 ex:cd/tmp/cd2022bg3

2.查看遠端倉儲路徑 git remote -v 如未設定過upstream只會跑出兩行

3.git remote add upstream https://github.com/409232xx/cd2022bgx.git把409232xx的cd2022bgx倉儲設置為自己的upstream，執行此指令後無返回任何訊息，再次執行指令 git remote -v 檢查是否成功。

4.git status 檢查本地已沒有未提交之修改

5.git fetch upstream 抓取409232xx倉儲更新

6.git checkout main 切換至main分支

7.git merge upstream/main 合併遠程main分支

8.git push 將本地倉儲推送至fork409232xx後自己名下之倉儲
